Crédit Start ATB

La solution idéale pour devenir propriétaire sans contraintes financières

Vous voulez passer votre permis de conduire, mais le coût de la formation vous freine ?

Avec START de l’ATB, financez jusqu’à 2000 DT pour votre permis moto, auto ou poids lourd.

Pour qui ?

Start est pour :

  • Toute personne souhaitant passer son permis de conduire (auto, moto, poids lourd).
  • Les jeunes de 18 à 35 ans.
  • Les jeunes actifs.

Quel type de permis ?

  • Permis auto
  • Permis moto
  • Permis poids lourd

Atouts

Un financement jusqu’à 2000 DT.
Une durée de remboursement pouvant atteindre 36 mois

Plus de liberté

Avec votre permis en poche, offrez-vous la liberté de bouger où vous voulez, quand vous voulez

Concentrez-vous sur le volant, START s’occupe du reste
